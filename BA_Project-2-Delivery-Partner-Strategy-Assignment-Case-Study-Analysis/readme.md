# Delivery Partner Strategy Assignment Case Study Analysis

## Project Overview

This project focuses on improving delivery partner performance and operational efficiency through two key initiatives:

1. Designing a cost-effective incentive program to increase partner availability and supply-hour contribution.
2. Developing a Partner Quality Evaluation Framework to measure partner performance and identify improvement opportunities.

The analysis demonstrates how data-driven decision-making can improve workforce management, service quality, and operational performance while maintaining cost efficiency.

---

## Business Objectives

The primary objectives of this project were:

* Increase delivery partner participation and platform availability.
* Encourage contribution during business-critical periods.
* Maintain Cost Per Task (CPT) below the target threshold of ₹0.90.
* Develop a structured Partner Quality Evaluation Framework.
* Identify partners requiring coaching or retraining.
* Improve operational efficiency and customer satisfaction.
* Support workforce management through analytical insights.

## Part 1 – Incentive Program Design

A points-based incentive framework was developed to encourage higher supply-hour contribution while maintaining financial sustainability.

### Key Features

* Supply-hour based reward structure.
* Day-priority multipliers for critical business days.
* Cost-efficient incentive allocation.
* Performance-driven participation model.

### Key Result

| Metric     | Value           |
| ---------- | --------------- |
| Target CPT | ₹0.90           |
| Actual CPT | ₹0.37           |
| Status     | Target Achieved |

The proposed framework successfully remained below the company's cost threshold while encouraging increased workforce availability.

---

## Part 2 – Partner Quality Evaluation Framework

A weighted scoring model was developed using operational and customer service metrics.

### Performance Categories

* Excellent
* Good
* Needs Improvement
* Retraining Required

### Partner Performance Summary

| Category            | Partners |
| ------------------- | -------: |
| Excellent           |    2,971 |
| Good                |      523 |
| Needs Improvement   |       14 |
| Retraining Required |       45 |
| Total               |    3,553 |

The framework provides a scalable approach for ongoing partner performance monitoring and workforce optimization.

---

## Key Business Insights

* The incentive framework achieved a CPT of ₹0.37, significantly below the target threshold of ₹0.90.
* Approximately 84% of evaluated partners were classified as Excellent performers.
* Only 59 partners required targeted improvement initiatives.
* The quality evaluation framework successfully identifies operational risks and performance gaps.
* Data-driven workforce management can improve service quality while maintaining operational efficiency.

---

## Recommendations

* Implement the proposed incentive framework across delivery operations.
* Continuously monitor Cost Per Task (CPT) to maintain cost efficiency.
* Adopt the Partner Quality Evaluation Framework as an ongoing performance management tool.
* Reward high-performing partners to improve engagement and retention.
* Provide coaching support for partners requiring improvement.
* Conduct retraining programs for partners classified as Retraining Required.
